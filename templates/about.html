SecureVault is a Capstone Project 2 submission by the Innovative Group at Gateway Business College, Australia.

📖 Project Overview

SecureVault is a full-stack web application developed as part of the Bachelor of Information Technology (Cybersecurity) program at Gateway Business College, Sydney, Australia. The project addresses the widespread problem of weak password practices by providing real-time password strength analysis, secure encrypted storage, and an intuitive user interface — all in a single, locally-hosted application.


The system was designed to align with OWASP Top 10 security guidelines, NIST SP 800-63B recommendations, and academic research on password strength estimation. The companion browser extension (developed separately) connects to the same backend API, allowing credentials to be auto-filled directly in the browser.
