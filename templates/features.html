Everything You Need
to Stay Secure

SecureVault combines a powerful password strength checker with a secure, encrypted manager — all in one clean interface.

⚡
Real-Time Strength Analysis
As you type, SecureVault instantly analyses your password using the zxcvbn algorithm — checking for dictionary words, keyboard patterns, repeated characters, and more.
Core Feature
🎲
Password Generator
Generate cryptographically random 18-character passwords with a single click. Every generated password uses a full mix of uppercase, lowercase, numbers, and symbols.
Core Feature
🔒
Encrypted Vault Storage
Every password you save is encrypted using Fernet symmetric encryption before being written to the database. Raw passwords are never stored on disk.
Security
🛡️
Argon2id Master Password
Your master password is hashed using Argon2id — the winner of the Password Hashing Competition and the most resistant algorithm against GPU brute-force attacks.
Security
👥
Multi-User Support
Multiple users can create their own private vault on the same instance. Each user's passwords are completely isolated — no user can see another's credentials.
Multi-User
📋
One-Click Copy
Copy your username or password to clipboard instantly with the Copy User and Copy Pass buttons — no need to type or reveal passwords on screen.
Usability
🗂️
Category Organisation
Organise saved passwords into categories — Email, Banking, Shopping, Social, Work, and General — so you can find credentials quickly in a growing vault.
Organisation
🔍
Vault Health Dashboard
At a glance, see how many passwords are strong, weak, and how many categories you use. Identify and fix weak credentials before they become a problem.
Dashboard
🔌
Browser Extension Ready
All backend APIs are designed to connect directly to the companion browser extension. Auto-fill, save, and retrieve passwords from your browser without leaving the page.
Integration
